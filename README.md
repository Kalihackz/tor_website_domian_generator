Installation :
=============
* Clone or Download the repository.
* Unzip the zip.
* Open a terminal in the unzipped folder namely : `tor_website_domian_generator-main`
* Now enter the following commands in the terminal :

| Commands  |
| ------------- |
| chmod +x torterminal.sh |
